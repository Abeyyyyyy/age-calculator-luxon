# Age Calculator (Luxon + Flatpickr)

A simple age calculator built using HTML, CSS, and JavaScript.
This project uses **Luxon** for date calculations and **Flatpickr** for a better date picker UI.

## 🚀 Features

* Select birth date using a modern datepicker
* Calculate exact age (years, months, days)
* Input validation (empty & future date)
* Clean and responsive UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* [Luxon](https://moment.github.io/luxon/)
* [Flatpickr](https://flatpickr.js.org/)

## 📦 How to Run

1. Clone the repository:

```bash
git clone https://github.com/Abeyyyyyy/age-calculator-luxon.git
```

2. Open `index.html` in your browser
